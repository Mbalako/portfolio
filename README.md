- **Typing Effect** for hero role (Computer Science Student → Tech Enthusiast)

- **Interactive Elements**: Hamburger menu, form submission, card tilts

- **Performance Optimized**: Vanilla JS, no frameworks

## 🎨 Customization Guide

### 1. **Profile Photo**
Replace `.profile-placeholder` in `index.html`:
```html
<!-- Before -->
<div class="profile-placeholder">...</div>

<!-- After -->
<img src="your-photo.jpg" class="profile-img" alt="Maxwell Mbalako" />
```
Add `.profile-img` styles in CSS.

### 2. **Projects Section**
In `#projects .projects-grid`, replace placeholder cards:
- Add real screenshots
- Update titles, descriptions, tags
- Link to live demos & GitHub repos
- Example: [Card Structure](index.html#Lprojects)

### 3. **Social Links**
Footer `.footer-socials`:
```html
<a href="https://github.com/yourusername" class="social-link">GitHub</a>
<a href="https://linkedin.com/in/yourprofile" class="social-link">LinkedIn</a>
```

### 4. **Skills**
Update `.skill-card` data attributes in HTML/CSS:
```html
<div class="skill-pct" data-target="95">0%</div>
<div class="skill-bar" data-width="95"></div>
```

### 5. **Deploy**
- **GitHub Pages**: Push to `gh-pages` branch
- **Netlify**: Drag & drop folder
- **Vercel**: `vercel --prod`
